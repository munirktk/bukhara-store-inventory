Login Screen

A Login Screen After Launching The Store Inventory Mobile Application. User can login by clicking Login with Gmail account, using his gmail account. An alert will popup for choose account, user can add/select account to continue. Security alert will popup by Google. Click Advanced showing in the bottom of Alert 1 then click Go to Store Inventory showing in the bottom of Alert 2.

Welcome Screen

For users already signed in, a welcome screen where they can proceed with the current Gmail ID or switch accounts.

Home Screen

The home screen should have buttons for import, export, scan inventory, and exit app.

Import

Clicking on the Import button allows the user to send a request to their Google Drive and import a .txt file, Choose a .txt file to import.

Scan Inventory

Clicking on the Scan Inventory button allows the user to scan item seal numbers and update items with marked dates.

Export

Clicking on the Export button allows the user to export the modified file back to their Google Drive.
